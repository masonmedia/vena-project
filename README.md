A template of README best practices to make your README simple to understand and easy to use.

## Description

A working responsive webpage mockup for Vena Solutions. The site is built from scratch using Tailwind.css, custom css, and javascript.

## Libraries

I used AOS (animate on scroll), a simple and beautiful library for scrolling animation, in addition to Swiper.js, a modern javascript carousel plugin.
